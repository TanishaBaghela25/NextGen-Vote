Voting Process

How NEXTGEN VOTE Works

01
Register
Enter Aadhaar, EPIC ID, mobile, email. System checks for duplicates instantly.
02
Face Capture
Live webcam captures 100 face frames. AI detects, resizes, and stores as encrypted vectors.
03
OTP Verify
On election day, enter Aadhaar + EPIC. OTP sent to registered mobile for verification.
04
Face Match
AI compares live face with stored vectors using KNN. Only exact match proceeds.
05
Cast Vote
Select your party or NOTA. Vote is recorded with timestamp. Cannot vote again.
Face Recognition

Powered by
Computer Vision AI

Our system uses OpenCV Haar Cascade detection with K-Nearest Neighbor classification to match voter faces with sub-4500 distance threshold — ensuring only the registered voter can cast their ballot.

  • Detects face in real-time from webcam
  • 50×50 pixel face vector comparison
  • 100 training frames per voter for accuracy
  • Rejects multiple faces in frame
  • Cross-checks against full voter database
  • No spoof detection bypass possible
